Bienvenido


Ofrecemos nuestro producto a cualquier negocio, el cual consta de un checador de huella dactilar para poder tener un mejor control en un área en especifico, así mismo ofrecemos una aplicacion para celulares android ya que con está aplicación podran controlar y visualizar los registros de entrada y salida que obtiene nuestro checador, de igual manera contaran con una base de datos para almacenar información del dueño de dicha huella.


IMAGEN DEL CHECADOR


Somos un pequeño grupo de estudiantes universitarios que hemos desarrollado como proyecto, dicho checador para así aplicar las tecnologías de Internet de la cosas, la implementación de una base de datos, la creación de una app movíl y el desarrollo de una página web.
